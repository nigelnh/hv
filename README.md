# All-in-one run

```bash
python main.py
```

This executes 3 stages sequentially:
- Stage 0: `get_all_stock_data.py` - Get all stock data
- Stage 1: `calc_and_draw.py` - Calculate and draw historical volatility
- Stage 2: `ui.py` - Calculate financial ratios
